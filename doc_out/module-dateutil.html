(static) isDate(date) → {boolean}

Function is used to find object is date or not. Returns true when input type is date object otherwise false.
Parameters:
Name Type Description
date Date The js object
Returns:
True when input type is date object otherwise false.
Type
boolean
Examples
dateutil.isDate(new Date()); // return true
dateutil.isDate(undefined); // return false
dateutil.isDate(null); // return false
dateutil.isDate(1); // return false
dateutil.isDate("date"); // return false
var date = new Date();
dateutil.isDate(date); // return true
